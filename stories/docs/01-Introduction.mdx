# Inclusio Shoelace

Welcome to the Inclusio Shoelace design system documentation. This comprehensive design system is built on top of the [Shoelace](https://shoelace.style/) web component library, providing a complete set of reusable components, design tokens, patterns, and templates.

## What's Included

### Design Tokens
A complete token system covering:
- **Colors** - Primary, success, warning, danger, and neutral color palettes
- **Typography** - Font families, sizes, weights, and line heights
- **Spacing** - Consistent spacing scale for margins, padding, and gaps
- **Shadows** - Elevation system for depth and hierarchy

### Components
Over 50+ fully documented components including:
- Form controls (buttons, inputs, selects, checkboxes, etc.)
- Layout components (cards, dialogs, drawers, etc.)
- Navigation (tabs, menus, dropdowns, breadcrumbs)
- Feedback (alerts, badges, spinners, progress bars)
- And many more...

### Patterns
Common UI patterns for:
- Form layouts (login, signup, contact, profile)
- Navigation patterns (top nav, sidebar, mobile menu)
- Data display patterns
- Feedback patterns

### Templates
Ready-to-use page templates:
- Dashboard templates
- Settings pages
- Marketing pages
- Documentation layouts

## Features

### Light & Dark Themes
All components support both light and dark themes. Use the theme toggle in the Storybook toolbar to switch between themes.

### Accessibility
- WCAG 2.1 AA compliant
- Full keyboard navigation support
- Screen reader friendly
- Automated accessibility testing with axe

### Testing
- Visual regression testing
- Interaction testing
- Accessibility testing
- Cross-browser compatibility

## Getting Started

Browse the categories in the sidebar:
- **Design Tokens** - Explore the design token system
- **Components** - View all available components with interactive examples
- **Patterns** - See common UI patterns and how to implement them
- **Templates** - Check out complete page templates

## Usage

All components are web components and can be used in any framework or vanilla JavaScript:

\`\`\`html
<sl-button variant="primary">Click Me</sl-button>
\`\`\`

\`\`\`javascript
import '@shoelace-style/shoelace/dist/components/button/button.js';
\`\`\`
